# Учёт проектов и бонусов аналитиков

Отдельное веб-приложение для отслеживания запусков проектов, назначенных аналитиков и причитающихся им бонусов.

## Возможности

- Карточка проекта: название, дата запуска, аналитик, статус, бюджет, бонус, комментарий.
- Статусы: **Запущен → В работе → Завершён**, с возможностью **Отменён**.
- Бонусы считаются автоматически по ставке аналитика:
  - `% от бюджета проекта` (например, 5 % от бюджета), или
  - фиксированная сумма за проект.
- Можно вручную переопределить сумму бонуса для конкретного проекта.
- Статистика по аналитикам: количество проектов, начисленные и ожидаемые бонусы.
- Поиск по названию и фильтр по статусу.

## Запуск

Требуется PHP 7.4+.

```bash
php -S 0.0.0.0:8090 -t projects projects/router.php
```

После запуска приложение открывается по адресу `http://localhost:8090/`.

## API

Базовый префикс: `/api/`

| Метод   | Путь                  | Назначение                              |
|---------|-----------------------|------------------------------------------|
| GET     | `/api/analysts`       | Список аналитиков                        |
| POST    | `/api/analysts`       | Создать аналитика                        |
| PUT     | `/api/analysts/{id}`  | Обновить аналитика                       |
| DELETE  | `/api/analysts/{id}`  | Удалить (если нет проектов)              |
| GET     | `/api/projects`       | Список проектов с рассчитанным бонусом   |
| POST    | `/api/projects`       | Создать проект                           |
| PUT     | `/api/projects/{id}`  | Обновить проект                          |
| DELETE  | `/api/projects/{id}`  | Удалить проект                           |
| GET     | `/api/stats`          | Агрегированная статистика по аналитикам  |

### Пример: создание аналитика

```bash
curl -X POST http://localhost:8090/api/analysts \
  -H 'Content-Type: application/json' \
  -d '{"name":"Иван Петров","rate_type":"percent","rate_value":5}'
```

### Пример: запуск проекта

```bash
curl -X POST http://localhost:8090/api/projects \
  -H 'Content-Type: application/json' \
  -d '{
    "name":"Внедрение CRM в ООО Ромашка",
    "started_at":"2026-05-12",
    "analyst_id":"<id_аналитика>",
    "status":"launched",
    "budget":500000
  }'
```

## Правила расчёта бонуса

1. Если у проекта явно указан `custom_bonus` — используется он.
2. Иначе берётся ставка аналитика:
   - `percent` → `бонус = бюджет × ставка ÷ 100`
   - `fixed`   → `бонус = ставка`
3. У отменённых проектов бонус всегда `0`.
4. В статистике:
   - Завершённые → попадают в **«Начислено»**.
   - Запущенные / В работе → попадают в **«Ожидается»**.
   - Отменённые → нигде не учитываются.
